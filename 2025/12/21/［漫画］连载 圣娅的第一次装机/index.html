［漫画］连载 圣娅的第一次装机 Ep.1-40

作者：Sana

当前已更新40篇

1.预算的现实 (戒咖啡买 4090 -> 吃泡面)

2.全家福 (快递堆成山 -> 发现没买桌子)

3.静电妄想症 (摸哪都放电 -> 最后裸体装机)

4.心碎的声音 (压下 CPU 扣具时的“咔嚓”声)

5.那层膜 (装完散热器发现底下膜没撕)

6.内存暴力 (插不进去 -> 给主板做心肺复苏)

7.血祭 (IO 挡板割手 -> 性能提升 10%)

8.消失的螺丝 (掉进机箱 -> 进入异次元)

9.跳线地狱 (PWR_SW 插针 -> 熊掌般的手指)

10.显卡下垂 (4090 太重 -> 用手办顶着)

11.大力出奇迹 (24 针供电太硬 -> 差点把主板按断)

12.眼不见为净 (正面理线艺术 -> 背面依托)

13.心肺骤停 (风扇灯光全亮 -> 屏幕显示无信号)

14.铸币检测 (痛哭显卡坏了 -> 插主板 HDMI 上了)

15.心跳时刻 (装驱动时黑屏 -> 以为电脑死机了)

16.虚荣的巅峰 (跑分击败 99% -> 发完朋友圈索然无味)

17.电子斗蛐蛐 (3 万块的 4090 -> 最终只玩扫雷/小游戏)

18.显卡在唱歌 (啸叫滋滋作响 -> 戴上降噪耳机假装没听见)

19.洁癖的消亡 (第一天一粒灰都擦 -> 三个月后满是灰尘无所谓)

20.永远的韭菜 (装好顶配 4090 -> 5090 发布后看着手里变废铁）

21.连根拔起 (换硅脂 -> CPU 粘在散热器上被暴力拔出)

22.电源惊魂 (电脑不亮想退货 -> 发现电源开关没开)

23.内存单通道 (插在 1、2 槽 -> 性能减半 -> “难怪卡顿”)

24.Chrome 内存吞噬者 (64G 内存 -> 开 50 个网页 -> 电脑卡死)

25.Steam 龟速 (千兆光纤 -> 下载 2MB/s -> 骂运营商 -> 发现设置了限速)

26.静电膜未撕 II (显卡背板很热 -> 发现背板保护膜也没撕)

27.HDMI 没声音 (对着显示器喊话 -> 发现显示器没喇叭)

28.物理维修 (机箱共振 -> 一脚踹上去 -> 安静了)

29.AutoUpdate (Win11 强制更新 -> 关键时刻 -> 强制重启)

30.RGB 瞎眼 (晚上挂机下载 -> 机箱亮得像迪厅 -> 盖衣服睡觉)

31.屏幕挂灯智商税 (为了护眼 -> 照出一屏幕灰尘)

32.软路由的陷阱 (为了快 1ms -> 折腾 OpenWRT -> 全家断网三天)

33.ITX 的代价 (想装个小的 -> 手太大塞不进 -> 散热像烤箱)

34.线材定制坑 (为了好看买镀银线 -> 线序不对烧了硬盘)

35.人体工学椅 (花 5000 买椅子 -> 坐姿依然像个虾米)

36.水冷的诱惑 (觉得风冷吵 -> 上分体水 -> 漏水了！)

37.桌搭摄影师 (花一天理线摆拍 -> 拍完瞬间恢复猪窝)

38.NAS 焦虑症 (存照片 -> 怕硬盘坏组 RAID -> 买硬盘比照片还贵)

39.多屏邪教 (双屏不够 -> 三屏竖屏 -> 脖子扭伤)

40.机械键盘入坑 (一把退烧 -> 十把 -> 润轴工)
