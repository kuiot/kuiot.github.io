核心功能设置

  1. Enable Frame Gen
    ✅ 总开关：必须选中，关闭时所有功能均不生效

  2. Game Lib

    • 选择游戏渲染库（示例）：
      • libUE4.so（虚幻引擎游戏，如《幻塔》）
      • OpenGL/Vulkan（这些则根据游戏实际引擎进行选择）

        GPU 优化建议【仅限可切换引擎的游戏】：

        • Adreno GPU → 优先 Vulkan
        • Mali GPU → 优先 OpenGL

  1. Method（渲染方法）

    • GPU (ES3.0)：
      • 效果：重影明显
    • GPU Compute (ES3.2)：
      • 效果：类似重复帧
    • GPU+HW (A640+)：
      • 效果：光流法插帧（动态场景偶现伪影）
    • HW (A7xx+)：
      • 8Elite存在问题：OpenGL闪退/Vulkan闪屏，无法正常使用

  2. Preset（预设帧率）

    • 60→120：生成1帧（30→60同理）
    • 40→120：生成2帧（原生帧越低伪影越多）
    • Auto 60fps cap：原始帧会被限制在≤60及以下→并输出120帧
    • Auto 40fps cap：原始帧会被限制在≤40→并输出输出120帧
    • Auto Uncapped：似乎是固定插1帧（120→240）
    • Disable at high framerate：>60帧时停用插帧
    • +1 Uncapped：强制插1帧【其他+2、+3则固定插2、3帧】
    • Low latency：低延迟模式

  3. Overlay（性能监视器）

    • 选项：关闭/帧数/API/全部
    • 作用：调试时显示实时性能数据

  4. FPS Threshold（帧数阈值）

    • 功能：原生帧低于设定值时停用插帧
    • 原理：避免低帧放大瑕疵

  5. Quality Mode（画质档位）

    • Very Low：最低画质
    • Low：低画质
    • Medium：平衡模式
    • High：高画质

  6. FG Mode（插帧偏好）

  7. Present Mode（垂直同步）

    • Mailbox：
      • 优点：操作跟手性强
      • 缺点：易丢帧
    • FIFO：
      • 优点：画面流畅
      • 缺点：有时会出现延时/重影， 并且无法使用 Auto Preset 和 Frame Pacing

  8. Frame Pacing（帧同步）

    • 默认值：未知
    • 设为 6 可能改善表现

  9. Max frames in-flight（预处理帧）

    • 功能：增加 GPU 缓冲帧
    • 副作用：可能提升延迟

  10. Spoof GPU（GPU伪装）

    • 作用：模拟GPU型号
    • 现状：Adreno 830 无需启用

  11. GPU控制（频率管理）

    • 功能：强制升频
    • 实测：常规场景无需启用

典型问题场景

  1. 高速移动画面

    • 表现：画面撕裂 + 虚拟按键异常
    • 原因：FrameGen 基于最终渲染画面插帧

  2. 轻度伪影

    • 条件：中等画质 + 清晰度优先
    • 原理：插帧作用于 GPU 输出后

  3. 第一人称游戏
    ❌ 不适用：画面变化过快导致瑕疵倍增

  4. 视频插帧

    • 弹幕场景：
    • 纯净视频：

      解决方案：

      1. 关闭弹幕可获得优质体验
      2. 播放器支持：
        • VLC（需启用 OpenGL）
        • MX Player（部分版本不兼容）
