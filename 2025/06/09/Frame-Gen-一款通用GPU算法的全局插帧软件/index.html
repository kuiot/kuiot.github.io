发表于2025-06-09|更新于2025-06-09|手机高端优化

前置要求

  • 解锁 Bootloader
  • ROOT 方案：
    • KernelSU / KernelSU Next / SukiSU Ultra
    • Magisk

      注1：Apatch 未经测试，可用性不保证
      注2：经部分朋友测试，部分网游可能导致封号，需注意。

购买与激活

官方链接：官网链接

  • 支付方式：微信/支付宝
  • 价格：≈40 元（依据汇率浮动）

    未激活版有时间限制：每次启动仅支持 4分钟 插帧。

使用方法

  1. 安装官方 APK
  2. 刷入 Magisk 模块（官方提供）
  3. 安装完成后即可重启手机

核心功能设置

  1. Enable Frame Gen
    ✅ 总开关：必须选中，关闭时所有功能均不生效

  2. Game Lib

    • 选择游戏渲染库（示例）：
      • libUE4.so（虚幻引擎游戏，如《幻塔》）
      • OpenGL/Vulkan（这些则根据游戏实际引擎进行选择）

        对于可以切换渲染引擎的游戏可以优先选择以下方式【根据 PPSSPP 测试】：

        • Adreno GPU → 优先选择 Vulkan
        • Mali GPU → 优先选择 OpenGL

  1. Method（渲染方法）

    • GPU (ES3.0)：
      • 效果：重影明显
    • GPU Compute (ES3.2)：
      • 效果：类似重复帧
    • 以上效果均已过时，现在划分为 GPU 选项
    • GPU+HW (A640+)：
      • 效果：类似光流法插帧
    • HW (A7xx+)：
      • 8Elite存在问题：OpenGL闪退/Vulkan闪屏，无法正常使用

  2. Preset（预设帧率）

    • 60→120：生成1帧（30→60同理）
    • 40→120：生成2帧（原生帧越低伪影越多，并且会有更多的延迟）
    • Auto 60fps cap：原始帧会被限制在≤60及以下→并输出120帧
    • Auto 40fps cap：原始帧会被限制在≤40→并输出输出120帧
    • Auto Uncapped：似乎是固定插1帧（120→240）
    • Disable at high framerate：>60帧时停用插帧
    • +1 Uncapped：强制插1帧【其他+2、+3则固定插2、3帧】
    • Low latency：低延迟模式

  3. Overlay（性能监视器）

    • 选项：关闭/帧数/API/全部
    • 作用：调试时显示实时性能数据

  4. FPS Threshold（帧数阈值）

    • 功能：原生帧低于设定值时停用插帧
    • 原理：避免低帧放大瑕疵

  5. Quality Mode（效果档位）

    • Very Low：最低插帧效果
    • Low：低插帧效果
    • Medium：中等插帧效果
    • High：高插帧效果
    • 注：新版本新增百分比控制效果，可以做到更加精准的控制。

  6. FG Mode（插帧偏好）

  7. Present Mode（垂直同步）

    • Mailbox：
      • 优点：操作跟手性强
      • 缺点：易丢帧
    • FIFO：
      • 优点：画面流畅
      • 缺点：有时会出现延时/重影， 并且无法使用 Auto Preset 和 Frame Pacing

界面示意图：

  1. Frame Pacing（帧同步）

    • 默认值：未知
    • 设为 6 可能改善表现

  2. Max frames in-flight（预处理帧）

    • 功能：增加 GPU 缓冲帧
    • 副作用：可能提升延迟

  3. Spoof GPU（GPU伪装）

    • 作用：模拟GPU型号

  4. GPU控制（频率管理）

    • 功能：强制升频
    • 实测：常规场景无需启用

典型问题场景

  1. 高速移动画面

    • 画面扭曲 + 虚拟按键异常
    • FrameGen 基于最终渲染画面进行插帧，所以部分游戏中虚拟按键也会被当作插帧的画面。

  2. 轻度伪影

    • 中等画质 + 清晰度优先
    • 出现此伪影也基本可以确定，FrameGen的插帧是根据GPU生成后的画面进行推算插帧的

  3. 第一人称游戏
    ❌ 不适用：画面变化过快导致瑕疵倍增

  4. 视频插帧

  • 使用B站进行视频插帧，则需要关闭弹幕才可以获得良好效果
    • 弹幕场景：
    • 弹幕出现问题
    • 视频场景：
    • 视频出现问题

      解决方案：
      关闭弹幕可获得一个比较好的体验

  • 本地视频播放器支持：
    • VLC（需启用 OpenGL）
    • MX Player（部分版本不兼容）
    • REEX （经测试不可用）

各位可以测试一下每个插帧方法的效果。尤其是Adreno7xx系的朋友，可以试试HW的插帧方法效果如何，8Elite的Adreno830目前是用不了这个7xxHW的插帧方法。希望看后续更新能否解决这个问题。
