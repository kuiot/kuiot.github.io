隐私政策

最后更新日期：[2026年2月17日]

[K · 随心手记]（以下简称”我们”）非常重视用户的隐私。本隐私政策旨在说明当您使用我们网站上的 Giscus 评论功能时，我们如何收集、使用和保护您的信息。

1. 评论系统简介

本站使用 Giscus 作为评论系统。Giscus 是一个开源的评论系统，完全基于 GitHub Discussions 构建。这意味着所有的评论数据并非存储在我们的服务器上，而是存储在 GitHub 的服务器上。

2. 数据收集

当您在本站进行评论、回复或反应时，Giscus 会收集并传输以下信息至 GitHub：

  • GitHub 公开个人资料：包括您的 GitHub 用户名、头像以及您的公开 GitHub 个人资料信息
  • 评论内容：您发布的文字、图片、代码片段以及相关的元数据（如发布时间）
  • 技术数据：为了防止滥用，系统可能会记录您的 IP 地址（由 GitHub 自动处理，用于安全防护和反垃圾机制）

请注意：我们不会额外收集或存储您的评论数据在我们的私有数据库中。

3. 数据存储与处理

由于 Giscus 基于 GitHub Discussions：

  • 数据存储位置：您的评论将作为”讨论”存储在我们的 GitHub 仓库中。
  • 数据控制权：GitHub (Microsoft Corporation) 是上述数据的实际数据处理者。评论数据受 GitHub 的服务条款和隐私声明管辖

4. 第三方服务

使用本站评论功能涉及对第三方服务的使用。当您加载评论区或登录 GitHub 进行评论时，您的浏览器将与以下域名的服务器通信：

  • giscus.app：用于加载评论组件脚本
  • github.com 及 github.githubassets.com：用于身份验证、加载头像及存储评论数据
  • api.github.com：用于通过 GitHub API 获取评论数据

这些第三方服务提供商各自有其隐私政策，我们建议您查阅 GitHub 隐私声明 以了解他们如何处理数据。

5. Cookie 和本地存储

为了让评论系统正常工作，Giscus 会在您的浏览器中设置本地存储或 Cookie，包括但不限于：

  • 保持您的登录状态
  • 记住您的主题偏好（如暗色/亮色模式）

您可以通过浏览器的设置管理或删除这些 Cookie 和本地数据，但这可能会影响您在评论区的使用体验。

6. 数据安全

我们通过启用 HTTPS 加密传输来保护您的数据在传输过程中的安全。然而，互联网传输没有绝对的安全，我们无法保证传输过程的绝对安全性。

7. 政策变更

我们可能会不时更新本隐私政策。更新后的政策将在本页面公布，建议您定期查阅。
